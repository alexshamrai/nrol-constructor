# NROL Constructor

Training program configurator based on **The New Rules of Lifting Supercharged** by Alwyn Cosgrove & Lou Schuler (2013).

Plan your entire training year: configure 10 programs across 3 phases, select exercises filtered by your equipment, swap exercises within movement patterns, and build long-term training plans.

## Features

- **Exercise Catalog** -- 78 exercises across 10 movement patterns with descriptions (EN/UA)
- **Program Configurator** -- all 10 programs (BT I-IV, Hyp I-III, S&P I-III) with workout templates
- **Exercise Swap** -- swap exercises by pattern, direction, and equipment availability
- **Periodization** -- undulating (original) and linear modes for Hypertrophy programs
- **Wave Loading** -- S&P III wave loading tables (7-5-3 / 6-4-2 / 5-3-1)
- **Year Planner** -- build long-term plans with date calculations and timeline
- **Equipment Filter** -- mark your available equipment, see what's possible
- **Bilingual** -- full English and Ukrainian support
- **Export** -- PDF and XLSX export for configured programs
- **Offline** -- works without internet after initial load
- **Dark/Light theme**

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Docker

```bash

```
